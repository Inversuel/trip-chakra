# Next.js React Chakra ui

This repo contains project with infinite query on scroll + Chakra Ui

## Installation

Clone project 

```
git clone https://github.com/Inversuel/trip-chakra
```

Install depedency 
```
npm install 
```

To run app in dev mode

```
npm run dev
```

```
npm run serve
```
